# 版本信息

## v1.0.0 (2024-01-15)

### 功能特性
- ✅ 完整的N8N工作流自动化
- ✅ 双方案支持（标准/超低成本）
- ✅ 20+种生活小妙招主题
- ✅ AI文本、图像、语音生成
- ✅ 自动视频合成
- ✅ Google表格记录
- ✅ Discord通知

### 技术栈
- **自动化平台**: N8N
- **文本生成**: Groq API / Cohere API
- **图像生成**: ShuttleAI / HuggingFace
- **语音合成**: ElevenLabs / StreamElements
- **视频制作**: Creatomate / FFmpeg
- **存储记录**: Google Sheets
- **通知**: Discord

### 支持平台
- ✅ Linux (推荐)
- ✅ macOS
- ✅ Windows (WSL)
- ✅ Docker容器
